# 게시판 서비스

가장 기본적인 서비스인 게시판 기능을 둘러보는 서비스
스프링 부트 관련 기술들과 자바 17, 개발도구들을 경험해봤습니다.

## 개발 환경
* Intellij IDEA Ulrimate 2023.2.2
* Java 17
* Spring Boot 3.1.4

## 기술 세부 스택

Spring Boot

* Spring Boot Actuator
* Spring Web
* Spring Data JPA
* Rest Repositories
* Rest Repositories HAL Explorer
* Thymeleaf
* Spring Security
* H2 Database
* MySQL Driver
* Lombok
* Spring Boot DevTools
* Spring Configuration Processor
* QueryDSL 5.0.0
* Bootstrap 5.2.1
* Heroku
